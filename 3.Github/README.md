# Modern Glassmorphism Portfolio

A premium, responsive portfolio web application featuring a modern glassmorphism design and scroll-reveal animations.

## ✨ Features
- **Modern UI**: Sleek dark mode with gradient accents and glassmorphism cards.
- **Responsive Design**: Fully functional on desktop, tablet, and mobile.
- **Interactive Elements**: Custom hover effects, smooth scrolling, and animated form feedback.
- **Micro-animations**: Entrance animations for content sections.

## 🚀 Getting Started
1. Clone this repository.
2. Open `index.html` in any modern browser.

## 🛠️ Built With
- **HTML5**: Semantic structure and SEO optimization.
- **CSS3**: Custom properties, glassmorphism, and animations.
- **JavaScript**: Interactive features and DOM manipulation.
