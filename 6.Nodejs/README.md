# Node.js Static Website

This is a simple Node.js application using Express to serve a static website.

## How to Run Locally

### 1. Install Dependencies
Open your terminal in the project directory and run:
```bash
npm install
```

### 2. Start the Server
Run the following command to start the application:
```bash
npm start
```

### 3. Access the Website
Once the server is running, open your browser and go to:
[http://localhost:3000](http://localhost:3000)

---

## Folder Structure
- `server.js`: The main entry point for the Node.js application.
- `public/`: Contains the static files (HTML, CSS, images).
- `package.json`: Contains project metadata and dependencies.
- `deployment_guide.md`: Detailed instructions for deploying to AWS.
